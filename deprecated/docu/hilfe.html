Was sind die Favoriten? Favoriten sind Stationen, die Sie sich vorgemerkt haben. Vermutlich, weil Sie dort öfter losfahren.

Wie kann ich Favoriten hinzufügen? Dafür müssen Sie sich registrieren und angemeldet sein. Rufen Sie eine Station auf und klicken Sie auf einen der färbigen Sterne, neben dem Stationsnamen. In dieser Farbe erscheint dann auch Ihr Favorit.

Kann ich einen Favoriten nur für eine bestimmte Linie oder Richtung speichern? Nein, derzeit zeigt die App immer alle Linien in allen Richtungen an und speichert das auch so in den Favoriten. Wir arbeiten aber an diesem Feature… ;-)

Wie kann ich einen Favoriten löschen? Rufen Sie die Station über den entsprechenden Button auf und klicken Sie dann beim Stationsnamen auf den Mistkübel.

Kann ich die Favoriten umsortieren? Derzeit leider nicht. wir arbeiten aber daran … :-)

Wie kann ich beliebige andere Stationen aufrufen? Klicken Sie dazu im Menü ins Suchfeld und geben Sie den Namen Ihrer Station ein oder suchen Sie eine in der Liste.

Warum muss ich meine Position freigeben? Die App fragt nach Ihrer Position, um die Liste der Stationen im Suchfeld der Entfernung nach sortieren zu können. So sehen Sie immer, welche Station die nächstgelegene ist.

Was ist eine “Web App”? Eine Web App ist eine Webseite, die so eingerichtet ist, dass sie auch wie eine eigenständige App aussehen kann. Das ist für mobile Devices gedacht. Fügen Sie dazu ein Bookmark für Ihren Homescreen hinzu. Dann startet der Monitor nicht mehr im Browser, sondern eigenständig.
